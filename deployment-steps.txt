## 2. Environment Variables Setup
Create `.env.production` with these variables:
```env
# API URL for production
VITE_API_URL=https://your-meta-verified-app.vercel.app

# SMTP Configuration (Required for Email Notifications)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-specific-password

# Form One SMTP Config (if different from default)
FORM_ONE_SMTP_HOST=smtp.gmail.com
FORM_ONE_SMTP_USER=your-form-one-email@gmail.com
FORM_ONE_SMTP_PASS=your-form-one-app-specific-password

# Form Two SMTP Config (if different from default)
FORM_TWO_SMTP_HOST=smtp.gmail.com
FORM_TWO_SMTP_USER=your-form-two-email@gmail.com
FORM_TWO_SMTP_PASS=your-form-two-app-specific-password

# Admin Notification Emails
ADMIN_EMAIL=admin1@example.com
ADMIN_EMAIL_2=admin2@example.com
ADMIN_EMAIL_3=admin3@example.com
```

## 4. Deployment Steps
1. Go to https://vercel.com/new
2. Click "Import Git Repository"
3. Select your repository
4. Configure project settings:
   - Framework Preset: Vite
   - Root Directory: ./
   - Build Command: npm run build
   - Output Directory: dist
   - Install Command: npm install

5. Add Environment Variables:
   ```
   VITE_API_URL
   SMTP_HOST
   SMTP_PORT
   SMTP_USER
   SMTP_PASS
   FORM_ONE_SMTP_HOST
   FORM_ONE_SMTP_USER
   FORM_ONE_SMTP_PASS
   FORM_TWO_SMTP_HOST
   FORM_TWO_SMTP_USER
   FORM_TWO_SMTP_PASS
   ADMIN_EMAIL
   ADMIN_EMAIL_2
   ADMIN_EMAIL_3
   ```

6. Click "Deploy"

## 5. Post-Deployment Verification

2. Verify Meta Verified landing page loads correctly
3. Test Meta verification form submission
4. Check email notifications
5. Verify mobile responsiveness
6. Test form validation
